
이 영상은 키워드 검색 프로그램을 만드는 과정을 담을 예정입니다.

c# + WPF 언어 및 기술을 가지고 네이버/11번가/인터파크등의 api를 호출 하여 만들었습니다.

이번 프로그램을 만드는 과정을 따라하면서 개발의 문법을 습득하고 구조를 이해할수 있는 능력을 키웠으면 좋겠습니다.

최종 목표로 하는 프로그램을 보여준후 개발을 진행 하겠습니다.

(프로그램실행)

Search 와 api Info 탭으로 구성 하였고
api info에는 api 정보를 입력할수 있게 구성 하였습니다.
검색을 진행하겠습니다.
검색결과를 보면 검색어와 연관된 검색어들이 나오고 월간 검색수 및 클릭수 광고의 노출 상태를 불러올수있습니다.
분석을 눌러보면 네이버/11번가/인터파크의 등록된 상품수를 가지고 올수 있습니다.
모든 결과는 자동 저장 됩니다.
데모는 여기까지하고 
이제 검색 프로그램을 개발 진행 하겠습니다.

먼저 개발 환경 및 api정보를 발급 받겠습니다.


(프로젝트 생성)
